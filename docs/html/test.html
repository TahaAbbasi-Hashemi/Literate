Example Calculator Program

This program provides basic arithmetic operations: addition, subtraction, and multiplication. It is designed to demonstrate how to structure a simple calculator using Python functions and how to call these functions from a main script.

calculator

Calling: addition, subtraction, multiplication

Called by: 

# calculator.py

@call{addition}

@call{subtraction}

@call{multiplication}

Functions

Purpose: Adds two numbers. Parameters: a (int or float): The first number. b (int or float): The second number. Returns: The sum of a and b.

addition

Calling:

Called by: calculator

def add(a, b):
    return a + b

Purpose: Subtracts the second number from the first number. Parameters: a (int or float): The number to be subtracted from. b (int or float): The number to subtract. Returns: The difference between a and b.

subtraction

Calling:

Called by: calculator

def subtract(a, b):
    return a - b

Purpose: Multiplies two numbers. Parameters: a (int or float): The first number. b (int or float): The second number. Returns: The product of a and b.

multiplication

Calling:

Called by: calculator

def multiply(a, b):
    return a * b

Example Usage

main

Calling:

Called by: 

# main.py

from calculator import add, subtract, multiply

def main():
    num1 = 10
    num2 = 5

    print(f"{num1} + {num2} = {add(num1, num2)}")
    print(f"{num1} - {num2} = {subtract(num1, num2)}")
    print(f"{num1} * {num2} = {multiply(num1, num2)}")

if __name__ == "__main__":
    @call{main}
